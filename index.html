SOFTWARES

Types Of Softwares:

1. Application Softwares

2. Operating System Softwares

Application Softwares:

Application software is a type of computer program that performs a specific personal, educational, and business function.
Each application is designed to assist end-users in accomplishing a variety of tasks, which may be related to productivity, creativity, or communication.

Examples:

1. The Microsoft Suit: Excel, Powerpoint, Word, Office etc

2. Internet Browsers: Firefox, Chrome, Safari, Internet Explorer etc

3. Music Software: Pandora, Apple Music, Spotify etc

4. Communication Software: Slack, Skype, Zoom, Teams

Operating System Softwares:

An Operating System (OS) is a collection of software that manages computer hardware resources and provides common services for computer programs.
The operating system is the most important type of system software in a computer system.

Examples:

Microsoft Windows, Mac OS, Android OS, Linux, Ubuntu, Chrome OS, Fedora
